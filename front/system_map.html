🧠
AI Agent — 自主分析引擎
這是系統真正「AI 是核心」的部分。 AI 不只生成文字——它主動決定要查什麼資料、評估自己夠不夠確定、不確定就繼續調查。
🔄 Agent 循環（Claude 原生）
1. Claude 決定呼叫哪個工具
2. 工具執行 → 結果回傳給 Claude
3. Claude 評估：「夠確定嗎？」
4. 不夠 → 繼續呼叫更多工具
5. 夠了 → 呼叫 submit_analysis
📋 Gemini/GPT 模擬循環
Stage 1：基礎資料 → 初步評估
Stage 2：依需求收集額外資料
          → 修正評估
Stage 3：提交最終結論
          （含信心評分 0-100）

舊 AI 報告 vs 新 Agent
📄 舊：AI 報告模式
一次性：收集所有資料 → 丟給 AI → AI 說人話

🧠 新：Agent 模式
迭代式：AI 自主選工具 → 評估信心 → 不夠就繼續
AI 是決策者，拿掉 AI 這個功能就消失了
